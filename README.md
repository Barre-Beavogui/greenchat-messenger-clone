# GreenChat - clone de messagerie

Clone fonctionnel de type WhatsApp Web, code original, sans logo ni code proprietaire WhatsApp.

## Fonctionnalites

- discussions avec messages entrants/sortants;
- recherche et filtres: tous, non lus, groupes, favoris;
- statuts, appels et channels;
- creation de nouveau contact ou groupe;
- appels audio/video simules;
- message vocal simule;
- pieces jointes simulees;
- panneau d'informations contact;
- favoris, media locaux et journal d'appel;
- sauvegarde dans `localStorage`.

## Ouvrir

Double-cliquer sur `index.html`, ou ouvrir ce fichier dans un navigateur.

## Fichiers

- `index.html`: structure de l'application.
- `styles.css`: interface responsive.
- `app.js`: etat local, rendu, interactions et persistance.
